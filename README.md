# create_Bert_vocabulary
Get plain text in txt file and create BERT vocabulary


#Steps:

1- Open txt file and remove '\n' if exists.

2- run tokenizer.train to generate BERT voc

# Running

1-Submit job using:

`sbatch bert.srm`
